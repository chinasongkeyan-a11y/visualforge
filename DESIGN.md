# DESIGN.md

## 气质与意象

VisualForge 是一款代码驱动的视觉动画视频渲染平台。它的气质应该像一间深夜的渲染工作室——屏幕是唯一的发光源，黑暗中只有预览画面在闪烁。专业视频编辑软件（DaVinci Resolve、Premiere Pro）的深色界面是天然的参考锚点，但 VisualForge 不是桌面软件的移植，而是 Web 原生的创作工具，因此需要在"专业感"与"Web 轻量"之间找到平衡。

意象：深夜机房，显示器阵列中只有一块屏幕亮着，上面是正在渲染的动画帧。空气安静，只有风扇的低鸣。

## 视觉策略

- 编辑器界面采用深色主题（非纯黑，而是深蓝灰 #0d1117 ~ #161b22），与视频预览区形成层次
- 预览画布区域是视觉焦点，周围用暗色面板包裹，类似影院"黑场"效果
- 时间线轨道用不同色相区分（背景层冷色、内容层暖色、前景层亮色）
- 片段块用半透明色块 + 左侧色条标识类型，选中时高亮边框
- 整体不用花哨的渐变和阴影，保持工具的克制感

## 配色方案

编辑器界面（非视频内容）：
- 背景主色：`#0d1117`（GitHub Dark 风格，非纯黑，有微妙蓝调）
- 面板背景：`#161b22`（比主背景稍亮一级）
- 边框：`#30363d`（低对比度，不抢视觉焦点）
- 文字主色：`#e6edf3`（高对比度白）
- 文字次色：`#7d8590`（用于标签、时间刻度）
- 强调色：`#2f81f7`（GitHub Blue，用于选中态、按钮）
- 危险色：`#f85149`（用于删除、错误）

视频内容主题（Tech Blue）：
- 主色：`#0071e3`（苹果蓝）
- 背景亮：`#f5f5f7`，背景暗：`#1a1a2e`
- 文字：`#1d1d1f` / `#ffffff`

视频内容主题（Dark Mode）：
- 主色：`#00d4aa`（薄荷绿）
- 背景：`#0d1117` / `#161b22`
- 文字：`#e6edf3` / `#7d8590`

## 字体排版

编辑器界面字体：
- 使用系统字体栈（PingFang SC / Microsoft YaHei / sans-serif），无需加载外部字体
- 标题：font-weight 600，14px
- 正文：font-weight 400，13px
- 时间刻度：font-weight 400，11px，tabular-nums

视频渲染字体：
- 服务端：WenQuanYi Micro Hei（系统内置中文字体）
- 浏览器预览：系统无衬线字体栈
- 标题字号：64px（视频画布坐标系）
- 副标题字号：36px
- 正文字号：28px

## 动效与交互

- 片段拖拽时显示半透明幽灵预览
- 时间线缩放使用平滑过渡
- 播放头拖动时实时刷新预览（节流到 60fps）
- 属性面板字段变更后即时更新预览
- 渲染进度条使用线性增长，不做花哨动画
- 按钮悬停态使用 150ms ease 过渡

## 设计禁忌

- 不要在编辑器界面使用浅色/白色背景
- 不要使用紫色/粉色系装饰（保持工具感）
- 不要在工具栏使用图标以外的装饰性元素
- 不要在预览区周围使用边框装饰（让画面本身说话）
- 不要使用圆角大于 8px 的卡片（保持专业克制）
